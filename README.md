# Ploopy ChatBot

Ploopy is a desktop chatbot application that helps users manage tasks through a simple conversational interface.
You interact with Ploopy by typing commands into the input box and pressing Enter or clicking Send.

Ploopy is designed for users who prefer a fast, keyboard-friendly way to manage tasks without navigating complex menus.

Using the app:

1.Launch the app.  
2.Tpe a command in the text field at the bottom.  
3.Press Enter or click Send.  
4.Ploopy will reply in the chat window.
